==================
Tensorboard Logger
==================
Logging utilities to monitor training progress.
To view logged data run

.. code-block:: bash

    tensorboard --logdir <directory with logfiles>

in the terminal and open the URL (usually http://localhost:6006/) by holding control
and left clicking the link in the terminal.
